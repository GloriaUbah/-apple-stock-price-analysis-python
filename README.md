Apple (AAPL) Historical Stock Price Analysis

Overview

This project analyzes five years of Apple Inc. (AAPL) historical stock data using Python. The analysis focuses on data cleaning, transformation, feature engineering, time-series analysis, and visualization to uncover meaningful insights into Apple’s stock performance.

Dataset

* Apple Inc. (AAPL) Historical Stock Data
* Source: Yahoo Finance
* Period: 2021–2026

Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

Project Objectives

* Clean and preprocess stock data
* Perform time-series analysis
* Engineer new features
* Visualize trends
* Generate actionable insights

Features Created

* Daily Price Change
* Percentage Change
* Daily Returns
* 7-Day Moving Average
* 30-Day Moving Average
* Monthly Average Closing Price
* Rolling Volatility

Visualizations

* Apple Closing Price Trend
* Trading Volume Trend
* Moving Average Analysis
* Daily Percentage Returns
* Monthly Average Closing Price
* Rolling Volatility

Key Findings

* Apple’s stock demonstrated strong long-term growth.
* Trading volume increased during major market events.
* Moving averages clearly identified long-term trends.
* Daily returns fluctuated around zero.
* Volatility increased during periods of market uncertainty.
